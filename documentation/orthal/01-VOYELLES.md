# Les Voyelles Simples (Monophtongues)

## Vue d'ensemble

L'alsacien possède un système vocalique riche. ORTHAL utilise des accents pour distinguer les sons qui n'existent pas en allemand standard.

---

## Les Accents Distinctifs

### L'accent grave sur `i` → `ì`

Le graphème `ì` note un /i/ très ouvert, proche du /é/ français.

| Graphème | Son | Équivalent | Exemples |
|----------|-----|------------|----------|
| `i` | /i/ fermé | comme "mie" | `Fisch` (poisson) |
| `ì` | /i/ très ouvert | entre /i/ et /é/ | `nìt` (ne...pas), `mìt` (avec) |

**Paires distinctives** :
- `wisse` (blancs) ≠ `wìsse` (savoir)
- `schliffe` (aiguiser) ≠ `gschlìffe` (aiguisé)

---

### L'accent grave sur `a` → `à`

Le graphème `à` note un /a/ sombre, voilé, différent du /a/ clair.

| Graphème | Son | Équivalent | Exemples |
|----------|-----|------------|----------|
| `a` | /a/ clair | comme "patte" | `Jager` (chasseur) |
| `à` | /a/ sombre | plus grave | `Jàcht` (chasse), `Bàch` (ruisseau) |

**Paires distinctives** :
- `dànke` (remercier) ≠ `danke` (penser)
- `Mànn` (homme) ≠ `Manner` (hommes)

---

## La Lettre `e`

### En syllabe non accentuée
`e` représente le /e/ réduit (Murmellütt), comme dans "le" en français :
- `der Mànn` → le /e/ de "der" est réduit
- `làche` (rire) → le /e/ final est réduit

### En syllabe accentuée
`e` peut représenter :
- /é/ fermé long : `reda` (parler), `lega` (poser)
- /è/ ouvert bref (devant plusieurs consonnes) : `wette` (parier), `kenne` (connaître)

---

## La Lettre `ä`

Représente /è/ ouvert. S'utilise pour :

1. **Éviter les confusions** :
   - `känne` (pouvoir) ≠ `kenne` (connaître)
   - `Kälte` (le froid) ≠ `Kelte` (les Celtes)

2. **Conditionnel des auxiliaires** :
   - `sìe wär` (elle serait)
   - `sìe hätt` (elle aurait)

3. **Pluriels avec alternance vocalique** :
   - `Glàs` → `Gläser` (verres)
   - `Dorf` → `Därfer` (villages)

---

## Le Digramme `ee`

Représente /é/ fermé long. Emploi quasi systématique :

| Mot alsacien | Traduction | Allemand |
|--------------|------------|----------|
| `Meer` | mer | Meer |
| `scheen` | beau | schön |
| `See` | lac | See |
| `Schnee` | neige | Schnee |

---

## Les Lettres `u` et `ü`

| Graphème | Son | Équivalent | Exemples |
|----------|-----|------------|----------|
| `u` | /ou/ | comme "fou" | `Hund` (chien), `gut` (bon) |
| `ü` | /u/ | comme "tu" | `Hüss` (maison), `dü` (tu) |
| `ù` | /ou/ très ouvert | régional | `dùmm` (stupide), `kùmme` (venir) |

---

## La Lettre `ë`

Représente /e/ très ouvert (proche de "in" sans nasalité) :

| Mot alsacien | Traduction | Allemand |
|--------------|------------|----------|
| `dënne` | ce, cet | diesen |
| `Hëmet` | pays natal | Heimat |
| `Mënung` | opinion | Meinung |

---

## La Lettre `œ`

Représente un son proche de "un" français (sans nasalité) :

| Mot alsacien | Traduction |
|--------------|------------|
| `Dœ` | jour |
| `œnder` | autre |

---

## Le Digramme `oo`

Représente /o/ long. Utilisé principalement devant `ch` :

| Mot alsacien | Traduction | Allemand |
|--------------|------------|----------|
| `nooch` | près | nah |
| `Noochber` | voisin | Nachbar |
| `hooch` | haut | hoch |

---

## Résumé des Voyelles

| Graphème | Son | Brève | Longue |
|----------|-----|-------|--------|
| `a` | /a/ clair | ✓ | ✓ |
| `à` | /a/ sombre | ✓ | ✓ |
| `e` | /e/ réduit ou /é/ | ✓ | ✓ |
| `ë` | /e/ très ouvert | ✓ | ✓ |
| `ä` / `è` | /è/ ouvert | ✓ | ✓ |
| `i` | /i/ fermé | ✓ | ✓ |
| `ì` | /i/ très ouvert | ✓ | ✓ |
| `o` | /o/ | ✓ | ✓ |
| `ö` | /eu/ | ✓ | ✓ |
| `u` | /ou/ | ✓ | ✓ |
| `ù` | /ou/ très ouvert | ✓ | ✓ |
