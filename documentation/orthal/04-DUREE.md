# Durée des Voyelles

## Règles Fondamentales

ORTHAL s'appuie sur les lois de durée de la phonétique allemande.

### Les Trois Formules Clés

| Règle | Formule | Résultat |
|-------|---------|----------|
| Syllabe ouverte | V = VL | Voyelle **longue** |
| Voyelle + 1 consonne | V + C = VL | Voyelle **longue** |
| Voyelle + 2+ consonnes | V + CC = VB | Voyelle **brève** |

---

## Exemples Pratiques

### Voyelle longue (syllabe ouverte)
- `ga` (donner) → /a/ long
- `hà` (avoir) → /a/ long
- `sì` (être) → /i/ long

### Voyelle longue (+ 1 consonne)
- `Gràs` → /a/ long
- `hole` → /o/ long
- `Ros` → /o/ long (rose)

### Voyelle brève (+ 2 consonnes)
- `boll` → /o/ bref
- `Kàss` → /a/ bref
- `Ross` → /o/ bref (cheval)

---

## Cas Particulier : Devant `ch`

Le digramme `ch` compte comme **deux consonnes**. La voyelle précédente est donc **brève** par défaut :

| Mot | Voyelle | Traduction |
|-----|---------|------------|
| `ich` | brève | je |
| `Kràch` | brève | bruit |
| `Loch` | brève | trou |
| `noch` | brève | encore |

Pour indiquer une voyelle **longue** devant `ch`, on la double :

| Mot | Voyelle | Traduction |
|-----|---------|------------|
| `Bààch` | longue | ruisseau |
| `nooch` | longue | près |
| `Noochber` | longue | voisin |

---

## Exceptions : Mots Grammaticaux

Les mots grammaticaux courts font exception à la règle V + C = VL. Leurs voyelles restent **brèves** :

### Prépositions
`àb`, `àn`, `àm`, `bis`, `ìn`, `ìm`, `mìt`, `ùf`, `üs`, `vom`, `ùm`, `zùm`

### Adverbes
`nìt` (ne...pas), `nix` (rien), `hin un har` (en va-et-vient)

### Conjonctions
`un` / `ùn` (et), `äb` / `ob` (si), `bis` (jusqu'à), `wìl` (parce que)

### Auxiliaires (indicatif)
`bìn`, `sìn`, `hàb`, `hàn`, `hàt`, `het`

---

## Les Diminutifs

Pour les diminutifs en `-le`, `-la`, `-lì`/`-lé`, la durée de la voyelle se détermine sur le radical seul :

| Mot de base | Voyelle | Diminutif |
|-------------|---------|-----------|
| `Làd` (volet) | longue | `Ladla` (volet long) |
| `Glàs` (verre) | longue | `Gläsle` / `Gläsla` |
| `Nàs` (nez) | longue | `Nasle` / `Nasla` |

---

## Les Formes Conjuguées

La durée de la voyelle se détermine sur le **radical du verbe**, sans tenir compte des terminaisons.

### Exemple : `hola` (aller chercher)
Le radical `hol-` a une voyelle longue (V + C = VL).

| Forme | Voyelle |
|-------|---------|
| `ìch hol` | longue |
| `dü holsch` | longue |
| `gholt` | longue |

### Exemple : `rolla` (rouler)
Le radical `roll-` a une voyelle brève (V + CC = VB).

| Forme | Voyelle |
|-------|---------|
| `ìch roll` | brève |
| `dü rollsch` | brève |
| `grollt` | brève |

---

## Alternance de Durée dans la Conjugaison

Certains verbes alternent entre voyelle longue et brève :

### Exemple : `reda` (parler)
| Forme | Voyelle | Graphie |
|-------|---------|---------|
| Infinitif | longue | `reda` |
| 1ère pers. sing. | brève | `ìch redd` |
| Participe passé | brève | `greddt` |
| 1ère pers. plur. | longue | `mìr reda` |

### Exemple : `froga` (demander)
| Forme | Voyelle | Graphie |
|-------|---------|---------|
| Infinitif | longue | `froga` |
| 1ère pers. sing. | brève | `ìch frogg` |
| Participe passé | brève | `gfroggt` |
| 1ère pers. plur. | longue | `mìr froga` |
