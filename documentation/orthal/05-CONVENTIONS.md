# Conventions d'Écriture

## Majuscules et Minuscules

### Majuscule obligatoire

| Catégorie | Exemples |
|-----------|----------|
| Début de phrase | Après `.`, `?`, `!`, `:` |
| Noms communs (substantifs) | `Korb` (corbeille), `Hüss` (maison), `Litt` (gens) |
| Noms propres | `Schmidt`, `Charles` |
| Toponymes | `Mìlhüsa`, `Colmer`, `Strosbùrig`, `Elsàss` |
| Noms dérivés de lieux | `e Colmerer`, `e Elsasser`, `e Frànzos` |
| Verbes/adjectifs substantivés | `ebbis Scheens` (qqch de beau), `s Trìnke` (le boire) |
| Politesse | `Ìhr`, `Sìe`, `Èich` |

### Minuscule

| Catégorie | Exemples |
|-----------|----------|
| Articles | `der`, `de`, `d`, `di`, `s`, `e`, `a` |
| Démonstratifs | `die`, `dia`, `dàs`, `dìs` |
| Pronoms personnels | `ich`, `dü`, `ar`, `sìe`, `mìr` |
| Adjectifs | `scheen`, `bràv`, `wohr` |
| Adverbes | `garn`, `gschnall` |
| Prépositions | `ìn`, `üs`, `mìt`, `ohne` |
| Conjonctions | `un`, `àber`, `denn`, `wil` |
| Verbes | `schàffe`, `schriwe` |

## L'Apostrophe

### Contraction `m'r` et `d'r`
- `m'r` = `mìr` (nous, on)
- `d'r` = `der` (le, article)

### Verbes impersonnels avec `'s`
- `'s regnt` (il pleut) ← `es regnt`
- `'s schneit` (il neige)
- `'s dundert` (il tonne)

### Formes contractées
| Plein | Contracté |
|-------|-----------|
| `es ìsch` | `'s ìsch` |
| `ìn dem` | `ìm` |
| `àn dem` | `àm` |
| `zü dem` | `zum` |

### Quand omettre l'apostrophe ?
Pour l'article féminin/pluriel, ORTHAL recommande de ne pas utiliser d'apostrophe :
- ✅ `d Heimet` (le pays natal)
- ❌ `d'Heimet`

## Le Trait d'Union

### Mots composés longs
Pour faciliter la lecture des composés complexes :
- `Hochsproch-Wärterbuech` (dictionnaire de la langue standard)

### Coupure syllabique
| Règle | Exemple |
|-------|---------|
| Diminutifs | `Maid-la`, `Kin-del` |
| Composés | `Hochsproch-Wärterbuech` |

---

## Éviter le Triplement de Consonnes

Quand un composé produirait trois consonnes identiques consécutives :
- ❌ `Schifffahrt`
- ✅ `Schiff-Fahrt` ou `Schiffahrt`

## L'Article Défini

### Formes selon les régions

| Région | Masculin | Féminin/Pluriel |
|--------|----------|-----------------|
| 68 (HA, BAS) | `der` / `dr` / `d'r` | `d` |
| 67 (BAN, FR) | `de` | `d` |

### Exemples
- **68** : `der Mànn` (l'homme), `d Fràu` (la femme)
- **67** : `de Mànn` (l'homme), `d Fràu` (la femme)

---

## L'Article Indéfini

| Région | Forme | Exemple |
|--------|-------|---------|
| 68 | `a` | `a Mànn` (un homme) |
| 67 | `e` | `e Mànn` (un homme) |

---

## Le Pronom Impersonnel "on"

Plusieurs formes équivalentes :
- `mer` / `me` / `ma` (on)
- `m'r` (forme contractée)

Exemple : `Mer sait nìt` / `M'r sait nìt` (On ne sait pas)
