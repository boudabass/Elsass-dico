# Table des Consonnes

## Sons ↔ Graphèmes

| Graphème | Son (API) | Description | Exemples alsacien | Allemand (réf) | Notes |
|----------|-----------|-------------|-------------------|----------------|-------|
| `b` | [b] / [p] | Occlusive faible sourde | `Barg`, `Gràb`, `redde` | Berg, Grab, reden | Entre /b/ et /p/ |
| `p` | [p] | Occlusive bilabiale faible | `Poscht`, `Päckel` | Post, Päckchen | Souvent réf. allemand |
| `d` | [d] / [t] | Occlusive faible sourde | `dü`, `Hund`, `redde` | du, Hund, reden | Entre /d/ et /t/ |
| `t` | [t] | Occlusive alvéolaire faible | `Tier`, `rette`, `Ràtt` | Tier, retten, Ratte | Souvent réf. allemand |
| `g` | [g] | Occlusive vélaire faible | `Gàrte`, `Gàss`, `Wag` | Garten, Gasse, Weg | - |
| `k` | [k] | Occlusive vélaire forte | `Kàrte`, `Kàss`, `Hoke` | Karte, Kasse, Haken | V précédé longue |
| `ck` | [k] | Occlusive vélaire forte | `Mucka`, `Drack` | Mücke, Dreck | V précédé brève |
| `qu` | [kv] | - | `Qualla`, `Quatsch` | Quelle, Quatsch | - |
| `m` | [m] | Nasale bilabiale | `Mord`, `wàrm` | Mord, warm | - |
| `n` | [n] | Nasale alvéolaire | `Nascht`, `Winn` | Nest, Wein | - |
| `ng` | [ŋ] | Nasale vélaire | `làng`, `sìnga`, `Rìng` | lang, singen, Ring | - |
| `nk` | [ŋk] | Nasale + Occlusive | `Bànk`, `hanke`, `krank` | Bank, hängen, krank | - |
| `pf` | [pf] | Affriquée | `Pfaffer`, `rupfe`, `Zopf` | Pfeffer, rupfen, Zopf | - |
| `s` | [s] | Fricative sifflante | `Stros`, `gros`, `lase` | Straße, groß, lesen | V longue ou diphtongue |
| `ss` | [s] | Fricative sifflante | `Gàss`, `Ross`, `wìsse` | Gasse, Ross, wissen | V brève |
| `ß` | [s] | Fricative (alternative) | `Stroß`, `groß`, `heiß` | Straße, groß, heiß | Recommandé si réf. All. |
| `sch` | [ʃ] | Fricative chuintante | `Schluss`, `Fìsch` | Schluss, Fisch | - |
| `sp` | [ʃp] | Réf. allemand | `spot`, `Sproch`, `spàre` | spät, Sprache, sparen | Début de mot |
| `st` | [ʃt] | Réf. allemand | `Stàdt`, `Stein`, `stolz` | Stadt, Stein, stolz | Début de mot |
| `schp` | [ʃp] | Phonétique | `Kaschperla`, `Waschp` | Kasperle, Wespe | Milieu/fin de mot |
| `scht` | [ʃt] | Phonétique | `Kùnscht`, `Wùrscht` | Kunst, Wurst | Milieu/fin de mot |
| `tsch` | [tʃ] | Affriquée chuintante | `Kütsch`, `rùtsche` | Kutsche, rutschen | - |
| `ch` | [x] | Fricative ("Ach-Laut") | `Bàch`, `Loch`, `Kràch` | Bach, Loch, Krach | Racle la gorge |
| `ch` | [ç] | Fricative ("Ich-Laut") | `ich`, `licht`, `Kiche` | ich, leicht, Küche | Nord Alsace / Munster |
| `h` | [h] | Aspiré | `Hüss`, `Hund`, `Hüet` | Haus, Hund, Hut | Initial |
| `h` | ∅ | Allongement | `fàhre`, `Ohr`, `Suhn` | fahren, Ohr, Sohn | Muet |
| `j` | [j] | Semi-consonne | `Jàger`, `jo`, `nèiji` | Jäger, ja, neue | Évite hiatus |
| `f` | [f] | Réf. allemand | `fìnde`, `schàffe` | finden, schaffen | - |
| `v` | [f] | Réf. allemand | `Vàtter`, `Volk`, `viel` | Vater, Volk, viel | - |
| `w` | [v] | Fricative dentale sonore | `Wàsser`, `wo`, `Wunder` | Wasser, wo, Wunder | - |
| `x / chs` | [ks] | - | `Àx`, `Àchs`, `sechs` | Axt, Achse, sechs | - |
| `z` | [ts] | Affriquée dentale | `Zitt`, `zweitens` | Zeit, zweitens | V longue ou diphtongue |
| `tz` | [ts] | Affriquée dentale | `Schàtz`, `Wìtz`, `Kritz` | Schatz, Witz, Kreuz | V brève |

## Règles de Doublement


| Voyelle longue | Voyelle brève |
|----------------|---------------|
| `hole` (chercher) | `Bolle` (boule) |
| `reda` (parler) | `reddt` (parlé) |
| `froga` (demander) | `froggt` (demandé) |
| `lase` (lire) | `làsse` (laisser) |

---

## Règles `/s/`

| Contexte | Graphie | Exemple |
|----------|---------|---------|
| Après voyelle **longue** | `s` ou `ß` | `Stros` / `Stroß` |
| Après voyelle **brève** | `ss` | `Gàss`, `Kàss` |
| Après **diphtongue** | `s`, `ss` ou `ß` | `Geis` / `Geiss` / `Geiß` |
