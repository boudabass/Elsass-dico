# Table des Diphtongues

## Sons ↔ Graphèmes

| Graphème | Son (API) | Description | Exemples alsacien | Allemand (réf) | Notes |
|----------|-----------|-------------|-------------------|----------------|-------|
| `au` | [au] | /a/ vers /ou/ | `Baum`, `Frau`, `kaufe` | Baum, Frau, kaufen | Aperture variable |
| `ei` | [ai] | /a/ vers /i/ | `Bein`, `Heimet`, `kei` | Bein, Heimat, kein | Réf. allemand |
| `ai` | [ai] | /a/ vers /i/ | `Mai`, `Kaiser`, `Baim` | Mai, Kaiser, Bäume | Recommandé pour homonymes |
| `àui` | [ɒi] | /a/ sombre vers /i/ | `Bàuim`, `Fràui`, `tàuisig` | Baum, Frau, tausend | Réf. allemand `au` |
| `ài` | [ɒi] | /a/ sombre vers /i/ | `rài(h)`, `Rhàin`, `Strài(h)` | roh, Rhein, Stroh | Autres cas |
| `aui` | [oi] | /o/ vers /i/ | `Bauim`, `Fraui`, `tauisig` | Baum, Frau, tausend | Réf. allemand `au` (67) |
| `oi` | [oi] | /o/ vers /i/ | `Boim`, `Froi`, `Roim(h)` | Baum, Frau, Rahm | Autres cas |
| `éi` | [ei] | /e/ fermé vers /i/ | `dréi`, `fréi`, `Schréi` | drei, frei, Schrei | Munster / Sundgau |
| `èi` | [ɛi] | /e/ ouvert vers /i/ | `drèi`, `frèi` | drei, frei | - |
| `ëi` | [æi] | /e/ très ouvert vers /i/ | `Frëid`, `Hëimet` | Freude, Heimat | Réf. `eu`/`ei` allemand |
| `äi` | [æi] | /e/ très ouvert vers /i/ | `Mäidla`, `Träim` | Mädchen, Träume | Autres cas |
| `öi` | [øi] | /eu/ (eux) vers /i/ | `blöi`, `gröi`, `döisig` | blau, grau, tausend | - |
| `äu` | [œy] / [œi] | /œ/ vers /u/ ou /i/ | `Bäum`, `Fräu`, `Sundgäu` | Baum, Frau, Sundgau | Presque tous les cas |
| `eu` | [œy] | /œ/ vers /u/ | `Beuta`, `keusch` | Beute, keusch | Calque allemand |
| `ie` | [iə] | /i/ fermé vers /e/ réduit | `Bier`, `lieb`, `Liecht` | Bier, lieb, Licht | - |
| `ia` | [ia] | /i/ fermé vers /a/ | `Biar`, `liab`, `Liader` | Bier, lieb, Lieder | Variante (68) |
| `iè` | [iɛ] | /i/ fermé vers /e/ ouvert | `lièb` | lieb | - |
| `ìe` | [ɪə] | /i/ ouvert vers /e/ réduit | `grìese`, `lìeb`, `Lìedel` | grüßen, lieb, Liedchen | - |
| `üe` | [yə] | /u/ fermé vers /e/ réduit | `Büe`, `güet`, `Wüet` | Bube, gut, Wut | - |
| `üa` | [ya] | /u/ fermé vers /a/ | `Büa`, `güat`, `Wüat` | Bube, gut, Wut | Variante (68) |
| `ui` | [ui] | /ou/ fermé vers /i/ | `Ruina` | Ruinen | - |
| `ùi` | [ʊi] | /ou/ ouvert vers /i/ | `Bùi`, `nùi`, `Sùi` | Bau, neu, Sau | - |
